## 1 快速开始 { #start }

> 主条目：注册登录与初次进入服务器

首次进入服务器的玩家需要完成账号注册。注册成功后，服务器会在符合条件的常用登录环境中启用自动登录；若登录环境发生变化，则需重新输入密码。

| 命令 | 说明 |
| --- | --- |
| `/register 密码 确认密码` | 首次进入服务器时注册账号。推荐设置为便于记忆的 8 位数字密码。 |
| `/login 密码` | 在未触发自动登录时，手动登录已注册账号。 |
| `/changepassword 旧密码 新密码` | 修改账号密码。 |

!!! warning "密码提示"

    推荐使用 8 位数字密码并妥善保存。若登录环境发生变化，服务器可能要求重新输入密码。

## 2 基础功能 { #basic }

本节收录玩家日常使用频率较高的住宅、传送、邮件与经济命令。

### 2.1 传送与住宅

| 命令 | 用途 |
| --- | --- |
| `/sethome [名称]` | 在当前位置设置住宅点。 |
| `/home [名称]` | 传送至住宅点。 |
| `/delhome <名称>` | 删除指定住宅点。 |
| `/tpa <玩家>` | 请求传送到指定玩家身边。 |

| `/tpdeny` | 拒绝收到的传送请求。 |
| `/back` | 返回上一个位置；能否使用以实际权限为准。 |
| `/tprandom` | 随机传送至服务器允许探索的区域，通常可简写为 /rtp。 |

### 2.2 聊天与经济

| 命令 | 用途 |

| `/mail clear` | 清空已读邮件。 |

## 3 领地系统 { #claim }

服务器使用 SimpleClaimSystem 提供领地保护。领地以 Minecraft 区块为基本单位进行认领，每个区块的水平范围为 16×16 格。

### 3.1 区块领地机制

1. 站在需要保护的区块内，认领当前区块。
2. 相邻区块可以继续认领，并合并为同一片连续领地。
3. 领地边界始终沿区块边界划分，不能自由选择任意宽度或长度。
4. 基地较大时，应检查建筑、地下仓库和机器是否跨越到未认领区块。
5. 通过领地菜单管理成员、访客权限和区块范围。

!!! info "常用命令"

    `/claim`、`/claim list`、`/claim chunks`、`/claim main`、`/unclaim`。具体子命令与菜单内容以服务器当前版本为准。

## 4 回收商店 { #recycle }

回收商店用于回收玩家多余资源，并按照公示价格发放金币，是服务器基础经济体系的一部分。

### 4.1 使用方法

1. 前往服务器指定的回收商店区域。
2. 找到对应物品的回收入口。
3. 检查回收单位、数量和单价。
4. 完成出售后，检查金币余额。

### 4.2 价格机制

回收价格会结合物品产量、用途、刷取难度和服务器经济状况进行调整。

## 5 箱子商店 { #quickshop }

> 本功能由 QuickShop 提供。

箱子商店支持玩家自行出售或收购物品，是玩家间交易与服务器市场流通的主要方式。

### 5.1 创建商店

1. 在自己的领地内放置箱子。
2. 手持需要出售或收购的物品，保持潜行状态并与箱子交互。
3. 根据聊天栏提示输入商品单价。
4. 选择出售或收购模式，并按提示完成创建。
5. 出售商店需要把商品放入箱子；收购商店需要保证账户余额充足。

### 5.2 交易与管理

- 购买和出售通常通过左键或右键箱子完成。
- 创建前应确认单价和每次交易数量。
- 商店应位于自己的领地内，同时为顾客保留必要的交互权限。
- 拆除商店前应先清空库存，并确认商店已正确移除。

### 5.3 常用备用命令

| 命令 | 用途 |
| --- | --- |
| `/qs buy` | 将当前商店切换为收购商店。 |
| `/qs sell` | 将当前商店切换为出售商店。 |
| `/qs price <价格>` | 修改当前商店的商品单价。 |
| `/qs remove` | 移除当前正在查看或选中的商店。 |
| `/qs find <物品>` | 查找出售指定物品的商店；是否开放以服务器配置为准。 |

!!! warning "提示"

    正常情况下优先使用箱子交互。命令可在交互提示未出现或需要修改商店时使用。

## 6 粘液科技 { #slimefun }

> 本功能由 Slimefun 提供。

粘液科技无需安装客户端模组，即可体验机器、材料、能源、货运与自动化等扩展内容。

| 命令 | 说明 |
| --- | --- |
| `/sf guide` | 获取或打开粘液科技指南。 |

### 6.1 新手发展建议

- 优先查看基础机器与材料分类。
- 根据近期制作目标解锁研究项目。
- 从基础工作台和材料处理设备开始发展。
- 贵重机器应放置在完整覆盖的领地内。
- 大型自动化设施应遵守服务器性能规定。

## 8 每日签到 { #signin }

> 签到功能由 LiteSign 提供。

玩家可通过服务器设置的签到入口完成每日签到，并领取对应奖励。

- 每日签到通常只能领取一次。
- 奖励内容和连续签到规则以游戏内说明为准。
- 领取前应确保背包有足够空间。

## 9 常见问题 { #faq }

??? question "为什么我不能破坏方块或打开箱子？"

    你可能位于他人的领地、出生点保护区或特殊功能区域。离开该区域，或联系领地主人检查成员权限。

??? question "为什么无法使用住宅点或传送命令？"

    请检查命令拼写、传送冷却、当前位置限制和自身权限。部分功能可能在特定世界或活动中暂时关闭。

??? question "箱子商店为什么无法交易？"

    常见原因包括库存不足、余额不足、商店关闭、领地阻止交互或交易方式不符合服务器设置。

??? question "粘液科技机器为什么不工作？"

    请检查机器结构、方块方向、输入材料、能源供应和所在区域权限。

??? question "基岩版菜单显示异常怎么办？"

    可尝试重新进入服务器，并记录发生位置、操作步骤和截图，在官方群内反馈。
