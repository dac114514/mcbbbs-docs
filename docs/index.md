## 1 快速开始 { #start }

> 主条目：注册登录与初次进入服务器

首次进入服务器的玩家需要完成账号注册。注册成功后，服务器会在符合条件的常用登录环境中启用自动登录；若登录环境发生变化，则需重新输入密码。

| 命令 | 说明 |
| --- | --- |
| `/register 密码 确认密码` | 首次进入服务器时注册账号。推荐设置为便于记忆的 8 位数字密码。 |
| `/login 密码` | 在未触发自动登录时，手动登录已注册账号。 |
| `/changepassword 旧密码 新密码` | 修改账号密码。 |

!!! warning "密码提示"

    推荐使用 8 位数字密码并妥善保存。若登录环境发生变化，服务器可能要求重新输入密码。

## 3 领地系统 { #res }

服务器使用 Residence 领地插件。领地可保护建筑、机器和箱子，并允许领地主人按需开放玩家权限。实际可创建面积、数量、价格和可用旗帜以服务器当前配置与 `/res help` 提示为准。

### 3.1 创建第一块领地

1. 手持**木锄**，在要保护范围的两个对角位置依次左键、右键，完成领地选区。
2. 对需要完整保护上下空间的基地输入 `/res select vert`，将选区从顶部延伸到底部。
3. 输入 `/res select size` 查看选区大小；如服务器启用了领地价格，再输入 `/res select cost` 确认费用。
4. 输入 `/res create <领地名>` 创建领地。名称建议只使用英文、数字、短横线或下划线，便于输入和管理。
5. 创建后使用 `/res current` 确认脚下的领地，并用 `/res info <领地名>` 查看主人、区域和权限。

> 快速创建：站在基地中心可尝试 `/res auto <领地名> <半径>`。该命令会按你可用的最大范围自动创建领地；若受到面积、金币或领地数量限制，按游戏内报错调整范围即可。

### 3.2 成员、访客与常用操作

| 目的 | 命令示例 | 说明 |
| --- | --- | --- |
| 查看自己的领地 | `/res list` | 列出自己拥有或可管理的领地。 |
| 查看当前位置 | `/res current` | 确认当前脚下是否在领地内。 |
| 查看详情 | `/res info <领地名>` | 查看领地主人和当前旗帜设置。 |
| 允许成员建造 | `/res pset <领地名> <玩家> build true` | 只对指定玩家开放建造；不要轻易对陌生人设置。 |
| 允许成员使用设施 | `/res pset <领地名> <玩家> use true` | 用于工作台、机器等交互；部分服务器还需单独开放容器权限。 |
| 允许成员打开容器 | `/res pset <领地名> <玩家> container true` | 适合可信成员共享箱子和设备。 |
| 设置传送点 | `/res tpset <领地名>` | 在当前站立位置设置该领地的传送位置。 |
| 传送至领地 | `/res tp <领地名>` | 能否使用以及冷却时间以服务器配置为准。 |
| 删除领地 | `/res remove <领地名>` | 删除前请确认建筑、箱子和机器已妥善处理；部分版本会要求再输入 `/res confirm` 确认。 |

!!! warning "权限提示"

    `/res set <领地名> <旗帜> true` 会把权限开放给所有访客；例如不应随意给陌生访客开放 `build` 或 `container`。与其开放公共权限，更推荐使用 `/res pset` 仅授权给可信成员。

参考：[Residence 官方命令说明](https://www.zrips.net/residence/commands/)；如命令参数与本文不同，请以服务器内的 `/res help` 为准。

## 4 回收商店 { #recycle }

回收商店用于回收玩家多余资源，并按照公示价格发放金币，是服务器基础经济体系的一部分。

### 4.1 使用方法

1. 打开钟表菜单，进入**回收商店**。
2. 在菜单中选择需要出售的物品，并检查回收单位、数量和单价。
3. 使用一键回收完成出售，再检查金币余额。

!!! info "基岩版快捷表单"

    基岩版玩家可在钟表菜单中打开回收商店专有表单，一键回收物品，无需输入命令。

### 4.2 价格机制

回收价格会结合物品产量、用途、刷取难度和服务器经济状况进行调整。

!!! warning "价格说明"

    具体回收清单和单价以当周公示数据为准，可能根据实际情况波动。

[查看 7/11–7/17 回收商店价目表](recycle-prices.md)

## 5 箱子商店 { #quickshop }

> 本功能由 QuickShop 提供。

箱子商店支持玩家自行出售或收购物品，是玩家间交易与服务器市场流通的主要方式。

### 5.1 创建商店

1. 在自己的领地内放置箱子。
2. 手持需要出售或收购的物品，保持潜行状态并与箱子交互。
3. 根据聊天栏提示输入商品单价。
4. 选择出售或收购模式，并按提示完成创建。
5. 出售商店需要把商品放入箱子；收购商店需要保证账户余额充足。

### 5.2 交易与管理

- 购买和出售通常通过左键或右键箱子完成。
- 创建前应确认单价和每次交易数量。
- 商店应位于自己的领地内，同时为顾客保留必要的交互权限。
- 拆除商店前应先清空库存，并确认商店已正确移除。

### 5.3 常用备用命令

| 命令 | 用途 |
| --- | --- |
| `/qs buy` | 将当前商店切换为收购商店。 |
| `/qs sell` | 将当前商店切换为出售商店。 |
| `/qs price <价格>` | 修改当前商店的商品单价。 |
| `/qs remove` | 移除当前正在查看或选中的商店。 |
| `/qs find <物品>` | 查找出售指定物品的商店；是否开放以服务器配置为准。 |

!!! warning "提示"

    正常情况下优先使用箱子交互。命令可在交互提示未出现或需要修改商店时使用。

## 6 粘液科技 { #slimefun }

粘液科技（Slimefun）无需安装客户端模组，即可体验机器、材料、能源、货运与自动化等扩展内容。具体配方、研究条件与机器功能以服务器内粘液科技指南为准。

| 操作 | 说明 |
| --- | --- |
| `/sf guide` | 获取或打开粘液科技指南。 |
| 查看分类 | 从基础材料、机器和能源分类开始，确认下一阶段目标。 |
| 解锁研究 | 根据近期制作目标解锁研究项目；缺少材料时可在指南中查看配方。 |

### 6.1 新手发展建议

1. 先从基础工作台、材料处理和常用资源开始，不必急于制作高阶机器。
2. 建立稳定电力后，再逐步扩展机器和货运设施。
3. 贵重机器应放在完整覆盖的领地内；大型自动化装置请遵守服务器性能规定。
4. 配方名称、汉化名称与外部 Wiki 可能存在差异，优先以游戏内指南显示为准。

## 9 常见问题 { #faq }

??? question "为什么我不能破坏方块或打开箱子？"

    你可能位于他人的领地、出生点保护区或特殊功能区域。离开该区域，或联系领地主人检查成员权限。

??? question "为什么无法使用住宅点或传送命令？"

    请检查命令拼写、传送冷却、当前位置限制和自身权限。部分功能可能在特定世界或活动中暂时关闭。

??? question "箱子商店为什么无法交易？"

    常见原因包括库存不足、余额不足、商店关闭、领地阻止交互或交易方式不符合服务器设置。

??? question "粘液科技机器为什么不工作？"

    请检查机器结构、方块方向、输入材料、能源供应和所在区域权限。

??? question "基岩版菜单显示异常怎么办？"

    可尝试重新进入服务器，并记录发生位置、操作步骤和截图，在官方群内反馈。

## 10 玩家行为规范 { #rules }

- 禁止使用作弊客户端、自动化脚本及未经许可的辅助工具。
- 禁止恶意破坏、盗窃、诈骗、骚扰或其他损害玩家正常体验的行为。
- 禁止利用漏洞复制物品、金币、技能经验或获取其他不当利益。
- 禁止刷屏、辱骂、引战及发布不适宜内容。
- 禁止建设明显影响服务器性能的大规模装置或长期运行的高负载设施。
- 发现漏洞后应立即停止使用，并及时向管理团队报告。
