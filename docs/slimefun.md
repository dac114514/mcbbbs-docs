# 粘液科技

粘液科技（Slimefun）无需安装客户端模组，即可在服务器中体验机器、材料、能源、货运与自动化等扩展内容。具体配方、研究条件与机器功能以服务器内粘液科技指南为准。

## 快速开始

| 操作 | 说明 |
| --- | --- |
| `/sf guide` | 获取或打开粘液科技指南。 |
| 查看分类 | 从基础材料、机器和能源分类开始，确认下一阶段目标。 |
| 解锁研究 | 根据近期制作目标解锁研究项目；缺少材料时可在指南中查看配方。 |

## 新手发展建议

1. 先从基础工作台、材料处理和常用资源开始，不必急于制作高阶机器。
2. 建立稳定电力后，再逐步扩展机器和货运设施。
3. 贵重机器应放在完整覆盖的领地内；大型自动化装置请遵守服务器性能规定。
4. 配方名称、汉化名称与外部 Wiki 可能存在差异，优先以游戏内指南显示为准。
